# 🎄 导出功能 - 小白使用指南

## 📋 你现在需要做什么（超简单3步）

### ✅ 第1步：找到你下载的文件
你已经点击了"Export"按钮，下载了一个文件：
- 文件名：`love-in-mexico-export.html`
- 位置：通常在"下载"文件夹里

### ✅ 第2步：合并照片数据（最重要！）

**用最简单的方式：**

1. **打开两个文件**（用任何文本编辑器，比如记事本、VS Code等）
   - 文件1：`dist/index-export.html`（在项目文件夹里）
   - 文件2：你下载的 `love-in-mexico-export.html`（在下载文件夹里）

2. **在文件2中**（你下载的文件）：
   - 按 `Ctrl+F`（Mac是 `Cmd+F`）搜索：`EMBEDDED_PHOTOS`
   - 找到这一行，应该类似这样：
     ```javascript
     window.EMBEDDED_PHOTOS = [{"id":"...","dataUrl":"data:image/..."}, ...];
     ```
   - **选中并复制**整个数组（从 `[` 开始，到 `]` 结束，包括中间所有内容）
     - 提示：这一行会非常长，包含所有照片数据

3. **在文件1中**（`dist/index-export.html`）：
   - 按 `Ctrl+F` 搜索：`EMBEDDED_PHOTOS`
   - 找到这一行：
     ```javascript
     window.EMBEDDED_PHOTOS = [];
     ```
   - **选中 `[]`**（两个方括号）
   - **粘贴**刚才复制的照片数据（替换掉 `[]`）

4. **保存文件1**（`dist/index-export.html`）

5. **重命名文件1**：
   - 将 `dist/index-export.html` 重命名为 `index.html`
   - 如果提示是否替换，选择"是"

### ✅ 第3步：使用导出版本

现在有两种方式：

**方式A：直接打开**
- 找到 `dist/index.html` 文件
- 双击打开（会在浏览器中打开）

**方式B：使用命令**
在终端运行：
```bash
npm run preview
```
然后在浏览器中打开显示的地址

## 🎉 完成！

现在你导出的网页就可以使用了：
- ✅ 照片已经嵌入，不需要重新上传
- ✅ 没有"Photo"按钮（照片已固定）
- ✅ 所有功能都正常（树上照片、CHAOS模式、点击放大）

## 💡 如果遇到问题

**问题1：找不到 `dist/index-export.html`**
- 解决：先运行 `npm run build`

**问题2：复制照片数据时出错**
- 解决：确保复制的是完整的数组，从 `[` 到 `]`，包括所有内容

**问题3：打开后没有照片**
- 解决：检查照片数据是否正确粘贴，确保 `window.EMBEDDED_PHOTOS = [...]` 这一行有内容

## 📁 文件位置说明

- `dist/index-export.html` - 构建后的模板文件（需要添加照片数据）
- `love-in-mexico-export.html` - 你导出的文件（包含照片数据）
- `dist/index.html` - 最终使用的文件（合并后的结果）
