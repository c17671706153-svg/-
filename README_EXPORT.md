# 📦 导出功能使用指南

## 简单三步使用导出的HTML文件

### 步骤 1: 导出HTML文件
1. 在网页中上传至少5张照片
2. 点击右下角的 **"💾 Export"** 按钮
3. 会下载一个名为 `love-in-mexico-export.html` 的文件

### 步骤 2: 构建项目
在终端中运行：
```bash
npm run build
```
这会生成 `dist` 文件夹，包含所有必要的文件。

### 步骤 3: 替换HTML文件
1. 打开 `dist` 文件夹
2. 将下载的 `love-in-mexico-export.html` 文件复制到 `dist` 文件夹
3. 重命名为 `index.html`（建议先备份原来的 `index.html`）

### 步骤 4: 使用导出版本
有两种方式：

**方式A：直接打开**
- 双击 `dist/index.html` 文件在浏览器中打开

**方式B：使用预览服务器**
```bash
npm run preview
```
然后在浏览器中打开显示的地址（通常是 http://localhost:4173）

## ✨ 导出版本特点

- ✅ **照片已嵌入**：无需重新上传，照片数据包含在HTML中
- ✅ **无Photo按钮**：无法修改照片（照片已固定）
- ✅ **功能完整**：树上显示照片、CHAOS模式照片墙、点击放大等功能都正常
- ✅ **可分享**：可以将整个 `dist` 文件夹分享给他人，他们可以直接使用

## 📁 文件结构

导出后的 `dist` 文件夹应该包含：
```
dist/
  ├── index.html          (你导出的HTML文件，已重命名)
  ├── assets/             (JavaScript和CSS文件)
  └── ...                 (其他资源文件)
```

## 💡 提示

- 导出的HTML文件包含所有照片的Base64数据，文件可能较大
- 如果需要修改照片，请使用正常版本重新上传并导出
- 导出版本适合分享给他人或部署到服务器
