# macOS App Icon

This directory should contain the app icon in the following formats:

- icon.icns (macOS icon format, required for building)
- icon.png (1024x1024, source format)

The icon should be a Christmas tree themed icon that matches the app's holiday aesthetic.

To create the icon.icns file from icon.png, you can use the following command on macOS:
```bash
sips -s format icns icon.png --out icon.icns
```

Or use online tools to convert PNG to ICNS format.

Recommended icon sizes for different resolutions:
- 16x16
- 32x32  
- 64x64
- 128x128
- 256x256
- 512x512
- 1024x1024
