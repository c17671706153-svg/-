# 🌐 多人使用指南 - 让其他人自定义照片并导出

## 场景说明

你想要让很多人使用你的网页：
1. 他们上传自己的照片
2. 导出包含他们自己照片的独立网页
3. 整个过程自动化，不需要手动操作

## ✅ 解决方案

我已经创建了自动化脚本，让整个过程变得非常简单！

## 📋 使用流程（对用户来说）

### 用户需要做的：

1. **访问你的网页**
   - 打开你部署的网页

2. **上传照片**
   - 点击 "📸 Photo" 按钮
   - 上传 5-20 张照片

3. **导出**
   - 点击 "💾 Export" 按钮
   - 下载 `love-in-mexico-export.html` 文件

4. **自动合并**（只需要运行一个命令）
   ```bash
   npm run merge-export
   ```
   或者如果他们已经构建过：
   ```bash
   npm run build
   npm run merge-export
   ```

5. **完成！**
   - 打开 `dist/index.html` 即可使用

## 🔧 技术实现

### 自动化脚本

我创建了 `scripts/merge-export.js` 脚本，它会：
1. 自动找到导出的HTML文件
2. 提取照片数据
3. 合并到构建后的HTML
4. 生成最终的 `index.html`

### 使用方法

```bash
# 1. 先构建项目
npm run build

# 2. 将导出的HTML文件放到 dist 文件夹
# （用户通过网页导出后，将文件复制到 dist 文件夹）

# 3. 运行自动合并脚本
npm run merge-export

# 完成！dist/index.html 已经包含照片数据
```
