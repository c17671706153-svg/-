# 🎄 圣诞互动相册

一个充满魔法的3D圣诞互动相册，让您在温馨的节日氛围中重温美好回忆。

## ✨ 特色功能

- 🎄 **3D圣诞树场景** - 逼真的3D圣诞树与节日装饰
- 📸 **照片互动展示** - 多张照片环绕圣诞树展示
- 🎵 **背景音乐** - 温馨的圣诞音乐营造节日氛围
- ✨ **粒子特效** - 飘落的雪花和闪烁的星光
- 🎁 **手势控制** - 支持触摸和鼠标交互操作
- 📱 **移动端适配** - 完美支持手机和平板设备

## 🚀 在线体验

[🌟 立即体验圣诞互动相册](https://your-app-url.vercel.app)

## 🎯 如何使用

1. 打开应用，欣赏3D圣诞树场景
2. 点击树上的装饰查看照片
3. 使用手势或鼠标控制视角
4. 享受背景音乐和节日特效
5. 分享您的圣诞回忆

## 🛠️ 技术栈

- **React 19** - 现代化前端框架
- **Three.js** - 强大的3D渲染引擎
- **React Three Fiber** - React的Three.js集成
- **Tailwind CSS** - 优雅的样式框架
- **Vite** - 快速的构建工具
- **MediaPipe** - 手势识别技术
